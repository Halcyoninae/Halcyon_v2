Halcyon-Properties File definitions

This window will show you the different usages for the properties files that can be used to modify the program. However this document does not cover every possible property that is avaliable with the program.

Program specific (user.)

  • user.default.folder: The default launch location of the select folder
  • user.force_optimization: Causes the program to optimize as much possible if you are facing overhead or are very limited
  • user.hidpi_value: The value to use for the Screen scaling property, BEST TO LEAVE AS IS

Player properties

  • audio.info.media_title_as_header: Use the internal media title (if avaliable) as the info header or the file name
  • audio.info.backdrop_use_greyscale: Use greyscale for the backdrop of the info display
  • audio.info.backdrop_gradient_style: Change how the gradient appears for the backdrop of the info display
  • audio.info.backdrop_use_gradient: Determines if the backdrop should use a gradient or not
